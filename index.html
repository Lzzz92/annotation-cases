管理员一次性配置步骤
  1. 在 GitHub 创建一个公开仓库（如 annotation-cases）
  2. 在仓库根目录新建文件 cases.json，内容填写 []，提交
  3. 进入仓库 Settings → Pages，Source 选 Deploy from a branch → main / root，保存
  4. 将 validator.html 也上传到仓库根目录，提交
  5. 去 GitHub Settings → Developer settings → Personal access tokens → Fine-grained tokens，生成一个 Token，权限勾选对应仓库的 Contents: Read and Write
  6. 将下方信息填好后「保存配置」；把 GitHub Pages 链接分享给标注员即可，标注员无需任何配置
